# 🛒 Store Data Analysis (Excel Project)

This project focuses on analyzing store sales data using Microsoft Excel to generate insights and support business decision-making.

---

## 📊 Objectives

* Analyze overall sales performance
* Identify top-performing products and categories
* Understand customer distribution (e.g., gender, region)
* Track monthly and seasonal trends
* Create interactive dashboards for better visualization

---

## 🛠️ Tools Used

* **Microsoft Excel**

  * Pivot Tables
  * Charts (Bar, Line, Pie)
  * Slicers & Filters
  * Conditional Formatting

---

## 📂 Dataset

The dataset includes:

* Order ID
* Product Category
* Sales Amount
* Quantity
* Order Date
* Customer Gender
* Region

---

## 📈 Analysis Performed

* Data Cleaning (handling missing values, formatting)
* Pivot Table Analysis
* Sales by Category & Region
* Monthly Sales Trends
* Customer Demographics Analysis

---

## 📊 Dashboard

The Excel file contains an interactive dashboard with:

* Top 5 Sales States
* Orders vs Sales
* Gender Distribution
* Order Channels

---


## ▶️ How to Use

1. Download the repository
2. Open `store_data.xlsx` in Microsoft Excel
3. Navigate through sheets:

   * Raw Data
   * Pivot Tables
   * Dashboard
